# DevOps Production Platform

A production-ready DevOps project built from the ground up using modern cloud-native technologies.

## Project Goals

- Build a production-ready Flask API
- Containerize applications with Docker
- Orchestrate services with Docker Compose
- Provision infrastructure using Terraform
- Deploy to AWS
- Deploy workloads to Kubernetes
- Build CI/CD pipelines with GitHub Actions
- Monitor applications using Prometheus and Grafana
